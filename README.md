# Unashamed Body V2 - Premium Redesign

This is a complete redesign of the [Unashamed Body](https://unashamedbody.com) patient journey.

## Features
- **"Concierge Medical" Aesthetic**: Luxury editorial design with Playfair Display and Inter.
- **Cinematic Animations**: Scroll-driven parallax and floating UI elements via Framer Motion.
- **Service Expansion**: Dedicated high-conversion pages for TRT, Weight Loss, Peptides, ED Treatment, and more.
- **CRM Persistence**: Integrated GoHighLevel (GHL) booking widgets and forms.
- **E-Commerce Loop**: Hard-linked CTAs to unashamedlabwork.com.
- **Mobile Optimized**: Priority-1 mobile patient experience.
- **Compliant**: Custom premium cookie consent banner included.

## Tech Stack
- Next.js 15+ (App Router)
- React 19
- TypeScript
- Tailwind CSS v4
- Framer Motion
- Lucide React

## Development
```bash
npm install
npm run dev
```
